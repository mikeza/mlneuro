mlneuro.preprocessing.stimulus.stimulus_at_times¶

mlneuro.preprocessing.stimulus.stimulus_at_times(stimulus_times, stimulus_data, select_times, stack_times=False)[source]¶

Get the value of timestamped data at the given times using linear interpolation over each dimension

Parameters:
  • stimulus_times (array-like shape = [n_samples,]) – The timestamps of stimulus data
  • stimulus_data (array-like shape = [n_samples, n_dims]) – The data at each sample
  • select_times (array-like shape = [n_select_samples,]) – The times to sample the stimulus at
Returns:

stimulus_data (shape = [n_select_samples, n_dims]) – The interpolated stimulus at the target times (over each dimension)
