mlneuro.multisignal.MultisignalSplit¶

class mlneuro.multisignal.MultisignalSplit(base_cv, *args, signal_arrays=None, **kwargs)[source]¶

Meta cross validator that takes multisignal (list) inputs and applies a base cross validator to each list item.

Parameters:
  • base_cv (type) – sklearn cross validator implementing split. If given an object, a new instance will not be created per signal and no collected args will be passed to it.
  • signal_arrays (dictionary) – arrays to split on construction of sub cross-validator e.g. signal_arrays={‘signal_based_data’: [arr1, arr2, arr3]} will pass the keyword signal_based_data=arr1 to signal1 and signal_based_data=arr2 to arr2, …
  • arguments (additional) – both additional arguments and keyword arguments will be collected and passed to the base_cv object
