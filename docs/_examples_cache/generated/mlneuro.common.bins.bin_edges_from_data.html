mlneuro.common.bins.bin_edges_from_data¶

mlneuro.common.bins.bin_edges_from_data(data, bin_count)[source]¶

Given a dataset, calculate a set of bin edges for each dimension that span the range of the data

Parameters:
  • data (shape (n_samples, n_dims)) –
  • bin_count (array-like, the number of bins per dimension. If scalar,) – repeated for n_dimensions
Returns:

  • bin_edges (a list of bin edges shape (n_bin_dim + 1,))
  • bin_count (a list of bins per dimension)
